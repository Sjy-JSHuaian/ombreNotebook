第四章：面向对象编程（OOP） 进阶
  • 三大特性：封装、继承、多态
  • 类定义：class 类名:，使用 __init__(self) 初始化
  • 创建对象：对象名 = 类名(参数)
  • 继承：class 子类(父类):，使用 super().__init__() 调用父类
  • 方法定义：第一个参数必须是 self
  • 类 vs 对象：类是模板，对象是实例
# 定义类
class CuteCat:
    def __init__(self, cat_name, age):
        self.name = cat_name
        self.age = age

    def speak(self):
        print("miao" * self.age)

# 创建对象
cat1 = CuteCat("咪咪", 3)

第五章：爬虫实战技巧 实战
  • 获取超链接：使用 BeautifulSoup 解析 HTML，soup.find_all("div", class_="class名")
  • 提取链接：a标签['href'] 获取超链接地址
  • 链接处理：注意相对路径需要拼接域名前缀
  • 爬取礼仪：设置合理间隔，尊重 robots.txt
  • 数据存储：使用字典存储 {标题: 链接}
  • 文档生成：使用 python-docx 创建 Word 文档
# 爬虫核心代码示例
from bs4 import BeautifulSoup
import requests

# 获取页面
url = "https://example.com"
response = requests.get(url)
soup = BeautifulSoup(response.text, 'html.parser')

# 查找所有链接
links_div = soup.find_all("div", class_="info_cate clearfix")

# 提取超链接
links_dict = {}
for div in links_div:
    a_tags = div.find_all('a')
    for a in a_tags:
        title = a.text.strip()
        link = a.get('href')
        links_dict[title] = link
# 生成Word文档
from docx import Document

doc = Document()
style = doc.styles['Normal']

# 添加标题和内容
doc.add_heading('爬虫结果', level=1)
doc.add_paragraph('这是爬取的内容...')
doc.save('result.docx')
🕷️ 爬虫要点总结：
1. 找到目标 div（class='info_txt clearfix'）
2. 定位到 dd 标签，再找所有 a 标签
3. 提取 href 属性获取链接
4. 注意处理相对路径（加域名前缀）
5. 使用 enumerate + zip 同时遍历标题和内容
6. 用 try-except 处理异常
